**从秘鲁河谷 8,000 年前的果壳，到全球年产 5,200 万吨的第一大坚果油料——花生的传奇**

---

花生，是五谷之外又一位来自美洲的"中国通"。它原产于南美洲安第斯山东麓，约 6,000–10,000 年前被驯化。明弘治年间传入中国东南沿海，清代普及为重要油料，20 世纪初山东大花生通过青岛港走向世界。今天，中国以年产近 2,000 万吨的规模稳居全球第一，占世界总产量近四成。从"长生果"到"早生贵子"，从花生油到花生酱，从许地山《落花生》的朴实哲思到 CRISPR 基因编辑的高油酸新种质——花生的故事，是农业史、贸易史与文化史的完美交织。

---

## 一、安第斯山东麓：花生的南美起源

### 物种起源

栽培花生（_Arachis hypogaea_ L.）是异源四倍体（2n=4x=40），由两个二倍体野生种 **Arachis duranensis**（A 基因组）和 **Arachis ipaensis**（B 基因组）自然杂交后染色体加倍形成。杂交事件发生的地理区域位于安第斯山脉以东，涵盖**玻利维亚东南部和阿根廷西北部**。

花生最独特的生物学习性是**地上开花、地下结果**（geocarpy）：开花授粉后，子房柄（peg）向下弯曲钻入土壤，在地下形成荚果。这一特性使花生能在沙质土壤中生长，也使其成为植物学中地下结果现象的经典案例。

### 最早考古证据

| 遗址/地点                                | 年代                               | 发现内容                                                                                   |
| ---------------------------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------ |
| **Nanchoc Valley, CA-09–77**（秘鲁北部） | 约 7,840 ± 40 BP（公元前 5840 年） | 花生果壳，经直接放射性碳定年，几乎肯定为栽培品种，因其出现在花生属自然分布区以外很远的地方 |
| **Quebrada Tierra Blanca**（秘鲁）       | 约 5,000 BP（公元前 3000 年）      | 最早确认为驯化种 _A. hypogaea_ 的遗存，果壳网状纹饰特征表明属于 hirsuta 变种               |
| **Huarmey Valley**（秘鲁沿海）           | 约 5,000 BP                        | 栽培花生遗存                                                                               |
| **Ancón**（秘鲁沿海，利马附近）          | 约 500–750 BC                      | 墓葬中出土大量陶罐，保存有花生——19 世纪最早引起学界关注的花生考古发现                      |

### 哥伦布前的分布

在欧洲人到达之前，花生已在美洲热带和亚热带地区广泛种植。1513–1524 年间，西班牙人 Oviedo y Valdés 在海地记录印第安人园圃中大量种植花生，并首次记录了美洲原住民名 **mani**（至今仍在古巴和西属南美洲使用）。阿兹特克语中花生的名称为 **tlalcacahuatl**（意为"土可可"）。

## 二、花生传入中国：明代海上丝路的馈赠

### 最早文献记载

中国最早记载花生的文献为 **1503 年（明弘治十六年）** 江苏《常熟县志》：

> "落花生，三月栽，引蔓不甚长。俗云花落在地，而子生土中，故名。霜后煮熟可食，其味绝美。"

随后，1504 年《上海县志》和 1506 年（正德元年）《姑苏县志》也出现了类似记载。

### 传入路线

| 路线               | 详情                                                                                                                         |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **葡萄牙人传入说** | 1516 年葡萄牙人登陆广州，即将花生传入中国。美国学者穆素洁指出，脆花生、砂糖及花生糖很快在全国成为寻常糖制品                  |
| **华侨传入说**     | 约在 16 世纪初叶，由华侨将花生种子从南洋引进福建、广东，然后逐渐引至他省。清初王凤九《汇书》明确指出"此神（花生）皆自闽中来" |
| **火者亚三传入说** | 正德十四年冬至十五年（1520 年），葡萄牙使团翻译火者亚三等人随明武宗南下南京，落花生很可能在此时被带到江南一带                |

> 关于宋元说：清代檀萃《滇海虞衡志》称落花生"宋元间……粤估从海上诸国得其种归种之"，但主流学界认为明代以前未有确凿的花生记载文献，宋元说可能系后人附会。

## 三、从零嘴到油料：明清花生的推广之路

### 明代（16–17 世纪）

明末，福建的泉州、漳州、兴化、邵武、汀州，广东的广州、潮州、高州、雷州、廉州、琼州等府均已种植花生。随后扩展到广西部分府县。花生在明代主要被当作**直接食用的零嘴**——煮熟或炒食。明末宋应星《天工开物》（1637 年）所列油料中**并无花生**，说明直到明末花生尚未被用作榨油。

### 清代：花生油登上历史舞台

花生作为油料的最早记载见于《三农纪》："炒食可果，可榨油，油色黄浊，饼可肥田。"乾隆年间，檀萃《滇海虞衡志》明确记载：

> **"落花生以榨油为上，故自闽及粤，无不食落花生油。"**

这表明当时福建、广东沿海地区已经普遍将花生用于榨油，花生油成为当地常见食用油。花生油是所有传统植物油中**诞生最晚的**，至清代才有正式记载，但发展迅速。

### 推广路线

| 阶段   | 时间                       | 地域                         |
| ------ | -------------------------- | ---------------------------- |
| 传入期 | 明弘治年间（1503–1520 年） | 江苏、上海、福建、广东沿海   |
| 扩展期 | 清初（17 世纪）            | 江南、安徽、江西、云南       |
| 普及期 | 清中后期（18–19 世纪）     | 北扩至山东、山西、河南、河北 |

## 四、山东大花生：近代中国农业的出口奇迹

### 美国大花生的引入

约 **1889 年**（光绪年间），美国传教士将 Virginia 型大花生引入山东蓬莱。1904 年，山东农桑总会在冠县、阳信、沂水、城武等地组织试种大花生。大花生含油量高达 40%–60%，在欧洲可作为橄榄油的廉价替代品，迅速赢得国际市场青睐。

### 青岛港的出口奇迹

| 年份    | 出口量                                       |
| ------- | -------------------------------------------- |
| 1908 年 | 9.5 万担（首次从青岛出口，目的地为法国马赛） |
| 1911 年 | 79.7 万担——三年增长 8 倍                     |

胶济铁路和津浦铁路建成后，山东花生大量汇集青岛出口，大花生从山东传播到全国。民国年间，山东博山县等低山丘陵区广泛种植花生，"花生出口，地瓜供食，较五谷获利稍厚"。花生与番薯在低山丘陵区形成了精巧的耕作制度——花生固氮恢复地力，番薯提供粮食。

## 五、现代中国的花生版图

### 产量与面积

中国是全球最大的花生生产国、消费国和进口国。2024 年全国花生种植面积约 **7,256 万亩**，总产量约 **1,898 万吨**，连续第二年增产。2025 年产量预估约 **2,000 万吨**，连续第三年增长。

| 年份 | 种植面积（万公顷） | 总产量（万吨）     |
| ---- | ------------------ | ------------------ |
| 2021 | 760                | 1,830              |
| 2022 | 约 465             | 约 1,833（减产年） |
| 2023 | 约 405             | 约 1,895           |
| 2024 | 约 484             | 约 1,898           |
| 2025 | 约 457             | 约 2,000（预估）   |

单产约 3,925 公斤/公顷，**超世界平均水平两倍**。

## 九、高油酸花生：育种革命

### 品种分类

中国花生按植物学特征分为五大类型：

| 类型                 | 特征                           | 代表品种         |
| -------------------- | ------------------------------ | ---------------- |
| 普通型（弗吉尼亚型） | 荚果普通形，果仁硕大饱满，晚熟 | 花育 22 号       |
| 珍珠豆型（西班牙型） | 荚果茧形，果仁圆形，早熟       | 适合食品加工     |
| 多粒型（瓦林西亚型） | 荚果串珠形，含 3–4 粒种子      | 含油量较高       |
| 龙生型               | 荚果多三仁果，种皮红色         | 中国最早种植类型 |
| 中间型               | 杂交育成品种，兼具过渡特征     | —                |

### 高油酸育种突破

普通花生油酸含量约 40%，高油酸花生可达 **75% 以上**，货架期延长 3 倍以上，有效预防心脑血管疾病。

- **冀花 19 号**：油酸含量近 80%，含油量超 55%，在河北推广面积达 136 万亩，连续两年入选国家主导品种
- **冀花 915**：油酸含量 82.3%，蛋白质含量 25.8%，2023 年创亩产 668 公斤高产纪录
- **商花 55 号**：油酸含量 79.2%，油亚比 21.18，小粒高油酸代表
- 河北省高油酸花生种植面积达 **215 万亩**，占全省花生种植面积的 59.6%，覆盖率全国第一

**前沿技术**：2025 年，中国工程院院士张新友团队利用 CRISPR/Cas9 基因编辑技术敲除花生 AhFAD2 基因，创制出**不含外源基因**的高油酸花生新种质。育种方向正从单一高油酸向"高油酸+"多元类型发展——花生酱加工型、榨油型、鲜食型等专用品种不断涌现。

## 十、花生与中国文化

### 从零嘴到国民食品

花生既是平民百姓的家常小食，又是宴席上的精致菜肴。加工方式多样：炒花生、煮花生、花生汤、花生糕、花生糖、花生碎、花生酥……至清末民初，花生已彻底成为全民食品，渗透到生活的方方面面。

### 文化符号

- **春节年货**：除夕夜全家围坐剥花生，寓意"团团圆圆"
- **新婚祝福**：花生撒在床上，寄托"**早生贵子**"的美好祝愿
- **长寿果**：营养价值被民间广泛认可，被视为养生佳品
- **金圣叹遗言**："花生米与豆腐干同吃，能嚼出火腿的味道"——成为中国文化史上最著名的临终遗言之一

### 许地山《落花生》

中国现代作家许地山的散文《落花生》以花生"地上开花、地下结果"的朴实特性比喻谦虚、不张扬的人格品质，成为影响几代中国人的经典名篇。文中父亲的教诲至今令人动容：

> "所以你们要像花生，它虽然不好看，可是很有用。"

## 结语：一粒花生里的全球史

从秘鲁 Nanchoc 河谷 8,000 年前的果壳，到安第斯山民手中"mani"的古老名称；从葡萄牙商船上的花生种子，到常熟县志中"落花生"的第一次中文记载；从乾隆年间闽粤"无不食落花生油"的普遍，到青岛港 1908 年首批出口欧洲的 9.5 万担；从许地山笔下"地上开花、地下结果"的朴实哲思，到 CRISPR 基因编辑创制的高油酸新种质——花生的传奇，是哥伦布大交换的华彩乐章，也是中国农业从传统走向现代的缩影。

今天，中国以年产 2,000 万吨位居全球第一，花生正从"长生果"的传统角色，走向高油酸、专用化、全产业链的新时代。一粒花生，曾经是安第斯山民的日常食物，如今是全球餐桌上的千亿产业。
